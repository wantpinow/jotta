# DS Research

Note: all these files are really rough and therefore prone to breaking.

### Processing

- `clean_vocab_csv.py`: Clean the 5000 word csv from Anki.
- `generate_vocab_variants.py`: Use gpt-3.5-turbo descriptions and lessons for each word.
- `embed_vocab.py`: Embed the vocab list using embedding models hosted on Modal.

### Analysis

- `analyse_vocab_frequency.py`: Analyse the frequency of words in the vocab list based on YouTube transcripts, news stories, and AI-generated text.
